## Beamer

Beamer とは, LaTeX でプレゼンテーションを作成するためのパッケージである. 
本記事では, 基本的な使い方しか扱わないため, 詳細に関しては以下を参照されたい.

- [latex-beamer.com](https://latex-beamer.com/)
- [Beamer Documentation](https://ftp.kddilabs.jp/CTAN/macros/latex/contrib/beamer/doc/beameruserguide.pdf)
- [Beamer (Overleaf Documentation)](https://www.overleaf.com/learn/latex/Beamer)

## 使い方

### frame 環境

Beamer では, スライド内容を `frame` 環境内に記述する. 

```latex
\begin{frame}[options]{Frame Title}{Frame subtitle}
  content
\end{frame}

% 以下でも同様.
\begin{frame}[options]
  \frametitle{Frame Title}
  \framesubtitle{Frame subtitle}
  content
\end{frame}
```

また, 簡単な内容であれば, `\frame` の利用も有効.

```latex
\frame{\titlepage}

% 以下と同様.
\begin{frame}
  \titlepage
\end{frame}
```

### block 環境

block 環境内に記述した内容は Beamer のテーマに合わせた箱で装飾される.

```latex
\begin{frame}
  \begin{block}{Block title}
    content
  \end{block}
\end{frame}
```

### 箇条書き

箇条書きには, `itemize` 環境や `enumerate` 環境を用いる.

```latex
\begin{frame}
  \begin{itemize}
    \item Item 1
    \item Item 2
    \item Item 3
  \end{itemize}
\end{frame}

% enumerate 環境によるカウンタの使用例
\begin{frame}
  \begin{enumerate}[1.]
    \item Item 1
    \item Item 2
    \item Item 3
  \end{enumerate}
\end{frame}
```

### 数式

数式には, `equation` 環境や `align` 環境等の数式環境を用いる.

```latex
\begin{frame}
  \begin{equation}
    \dfrac{\partial u}{\partial t} + u \dfrac{\partial u}{\partial x} = \nu \dfrac{\partial^{2} u}{\partial x^{2}}
  \end{equation}
\end{frame}
```

### 表

表には, `table` 環境と `tabular` 環境を用いる.

```latex
\begin{frame}
  \begin{table}[H]
    \centering
    \caption{First Table}
    \begin{tabular}{ccc}\hline
      A&B&C\\\hline
      D&E&F\\\hline
    \end{tabular}
  \end{table}
\end{frame}
```

### 図

図には, `figure` 環境を用いる.

```latex
\begin{frame}
  \centering
  \includegraphics[width=0.4\linewidth]{sample.png}
  \caption{First Figure}
\end{frame}
```

### アニメーション

Beamer でアニメーションを実現するには, `\pause` や `\uncover`, `<>` 等を用いる. 

#### `\pause`

```latex
\begin{frame}
  \begin{block}{First Block}
    content
  \end{block}
  \pause
  \begin{block}{Second Block}
    content
  \end{block}
  \pause
  \begin{block}{Third Block}
    content
  \end{block}
\end{frame}
```

アニメーションの遷移は, 

➀ First Block  
➁ First Block + Second Block  
➂ First Block + Second Block + Third Block

としたとき, ➀$\rightarrow$➁$\rightarrow$➂となる.

#### `\uncover`, `<>`

```latex
\begin{frame}
  \begin{block}{}
    \begin{itemize}
      \item<1-> One
      \item<2-> Two
    \end{itemize}
  \end{block}
  \uncover<3>{
    Three
  }
  \uncover<4>{
    Four
  }
\end{frame}
```

アニメーションの遷移は, 

➀ One  
➁ One + Two  
➂ One + Two + Three  
➃ One + Two + Four 

としたとき, ➀$\rightarrow$➁$\rightarrow$➂$\rightarrow$➃となる.

## 使用例

```latex:slide.tex
\documentclass[12pt, unicode]{beamer}
\usetheme{Madrid}
\usepackage{luatexja}
\usepackage{amsmath}
\usepackage{float}
\usepackage{url}
\newcommand{\mytoday}{\the\year/\the\month/\the\day}
\newcommand{\bhline}[1]{\noalign{\hrule height #1}}

\title{Beamer 入門}
\author{Eito YONEYAMA}
\date{\mytoday}
\institute[某大学]{某大学 情報系}

\begin{document}

\frame{\titlepage}

\begin{frame}{目次}
  \tableofcontents
\end{frame}

\section{Beamer とは}
\begin{frame}{Beamer とは}
  Beamer とは, LaTeX でプレゼンテーションを作成するためのパッケージである. 
  \newline
  \newline
  \textbf{関連リンク}
  \begin{itemize}
    \item \url{https://latex-beamer.com/}
    \item \url{https://ftp.kddilabs.jp/CTAN/macros/latex/contrib/beamer/doc/beameruserguide.pdf}
    \item \url{https://www.overleaf.com/learn/latex/Beamer}
  \end{itemize}
\end{frame}

\section{Beamer の使い方}
\begin{frame}{Beamer の使い方}
  本スライドでは, Beamer の基本的な使い方を, 記述例と出力例とともに紹介する. 今回扱う内容は以下の通り.
  \newline
  \begin{itemize}
    \item 箇条書き
    \item 数式
    \item 表
    \item 図
    \item アニメーション
  \end{itemize}
\end{frame}

\subsection{箇条書き}
\begin{frame}[fragile]{箇条書き 1/2}
  \begin{block}{Code}
    \begin{semiverbatim}
  \\begin\{itemize\}
    \\item Item 1
    \\item Item 2
    \\item Item 3
  \\end\{itemize\}
\end{semiverbatim}
  \end{block}
  \begin{block}{Output}
    \begin{itemize}
      \item Item 1
      \item Item 2
      \item Item 3
    \end{itemize}
  \end{block}
\end{frame}

\begin{frame}[fragile]{箇条書き 2/2}
  \begin{block}{Code}
    \begin{semiverbatim}
  \\begin\{enumerate\}[1.]
    \\item Item 1
    \\itme Item 2
    \\item Item 3
  \\end\{enumerate\}
\end{semiverbatim}
  \end{block}
  \begin{block}{Output}
    \begin{enumerate}[1.]
      \item Item 1
      \item Item 2
      \item Item 3
    \end{enumerate}
  \end{block}
\end{frame}

\subsection{数式}
\begin{frame}[fragile]{数式 1/2}
  \begin{block}{Code}
    \begin{semiverbatim}
  \\begin\{equation\}
    \\dfrac\{\\partial u\}\{\\partial t\} 
    + u \\dfrac\{\\partial u\}\{\\partial x\} 
    = \\nu \\dfrac\{\\partial\^\{2\} u\}\{\\partial x\^\{2\}\}
  \\end\{equation\}
\end{semiverbatim}
  \end{block}
  \begin{block}{Output}
    \begin{equation}
      \dfrac{\partial u}{\partial t} + u \dfrac{\partial u}{\partial x} = \nu \dfrac{\partial^{2} u}{\partial x^{2}}
    \end{equation}
  \end{block}
\end{frame}

\begin{frame}[fragile]{数式 2/2}
  \begin{block}{Code}
    \begin{semiverbatim}
  \\begin\{align*\}
    \&\\dfrac\{dx\}\{dt\}=ax-bxy\\\\
    \&\\dfrac\{dy\}\{dt\}=-cy+\\mu bxy
  \\end\{align*\}
\end{semiverbatim}
  \end{block}
  \begin{block}{Output}
    \begin{align*}
      &\dfrac{dx}{dt}=ax-bxy\\
      &\dfrac{dy}{dt}=-cy+\mu bxy
    \end{align*}
  \end{block}
\end{frame}

\subsection{表}
\begin{frame}[fragile]{表 1/2}
  \begin{block}{Code}
\begin{semiverbatim}
  \\begin\{table\}[H]
    \\centering
    \\caption\{First table\}
    \\begin\{tabular\}\{clr\}\\bhline\{1pt\}
      Centered\&Left-aligned\&Right-aligned\\\\\\hline
      A\&B\&C\\\\
      D\&E\&F\\\\\\bhline\{1pt\}
    \\end\{tabular\}
  \\end\{table\}
\end{semiverbatim}
  \end{block}
※\verb|\newcommand{\bhline}[1]{\noalign{\hrule height #1}}|
\end{frame}

\begin{frame}{表 2/2}
  \begin{block}{Output}
    \begin{table}[H]
      \centering
      \caption{First table}
      \begin{tabular}{clr}\bhline{1pt}
        Centered&Left-aligned&Right-aligned\\\hline
        A&B&C\\
        D&E&F\\\bhline{1pt}
      \end{tabular}
    \end{table}
  \end{block}
\end{frame}

\subsection{図}
\begin{frame}[fragile]{図 1/2}
  \begin{block}{Code}
\begin{semiverbatim}
  \\begin\{figure\}[H]
    \\centering
    \\includegraphics[width=0.4\\linewidth]\{santa.png\}
    \\caption\{素敵な雪だるま\}
  \\end\{figure\}
\end{semiverbatim}
  \end{block}
\end{frame}

\begin{frame}{図 2/2}
  \begin{block}{Output}
    \begin{figure}[H]
      \centering
      \includegraphics[width=0.4\linewidth]{santa.png}
      \caption{素敵な雪だるま}
    \end{figure}
  \end{block}
\end{frame}

\subsection{アニメーション}
\begin{frame}[fragile]{アニメーション 1/2}
  \begin{block}{Code}
\begin{semiverbatim}
  \\begin\{itemize\}
    \\item<1-> One
    \\item<2-> Two 
    \\item<3-> Three 
  \\end\{itemize\}      
\end{semiverbatim}
  \end{block}
\end{frame}

\begin{frame}{アニメーション 2/2}
  \begin{block}{Output}
    \begin{itemize}
      \item<1-> One
      \item<2-> Two 
      \item<3-> Three 
    \end{itemize}
  \end{block}
\end{frame}

\end{document}
```
